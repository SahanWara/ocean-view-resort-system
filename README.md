
# Ocean View Resort Reservation System

##  Project Description
The Ocean View Resort Reservation System is a web-based application developed to manage hotel reservations efficiently. It replaces manual booking systems to reduce errors and improve performance.

## Features
- User Login Authentication
- Add and Manage Reservations
- Billing System (based on room type and stay duration)
- Reports (total reservations)
- Help Section
- Input Validation

##  Technologies Used
- Java (Spring Boot)
- MySQL (Database)
- HTML, CSS, Bootstrap (Frontend)
- REST API

## System Architecture
Frontend → REST API → MySQL Database

## Functional Modules
1. Login System
2. Reservation Management
3. Billing System
4. Reports
5. Help and Validation
